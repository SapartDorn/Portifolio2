# Portfolio Pessoal

Este é meu portfolio pessoal desenvolvido com HTML, CSS e JavaScript.

## Como usar

1. Clone este repositório
2. Abra o arquivo `index.html` em seu navegador

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Funcionalidades

- Design responsivo
- Tema claro/escuro
- Animações suaves
- Formulário de contato
- Links para redes sociais
